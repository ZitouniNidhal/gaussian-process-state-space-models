# Advanced Topics

Advanced usage includes custom kernels, variational inference, and particle filtering.

- Use `gpssm.inference` helpers to build variational approximations.
- Extend `gpssm.models` for custom state-space dynamics.
- Compare filtering strategies using `gpssm.filters.KalmanFilter` and `gpssm.filters.ParticleFilter`.

These notes are a starting point for extending the package to more complex real-world systems.
